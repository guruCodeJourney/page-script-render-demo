this page show how defer script block the page render

Scripts with defer never block the page.

Scripts with defer always execute when the DOM is ready, but before DOMContentLoaded event. defer脚本会在文档渲染完毕后，DOMContentLoaded事件调用前执行。

The defer attribute is only for external scripts

在加载多个JS脚本的时候，async是无顺序的加载，而defer是有顺序的加载。
